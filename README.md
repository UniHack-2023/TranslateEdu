# TranslateEdu Documentation
 How to use:
 1. Install Python (we used 3.11.0)
 2. Install all libraries (from the requirements.txt) and upgrade pip:
    *python -m pip install --upgrade pip
    *pip install -r requirements.txt
3.Run the app
